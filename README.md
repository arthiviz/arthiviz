### 🐷 PigPeso:Pesagem Passiva de Suínos
 - Projeto construido com intuito de automatizar o processo de pesagem de suinos da suinocultura do IFPB campus Sousa, Foi construido este Site para monitoramento e registro do peso dos suinos durante todo sua vida para melhor acompanhamento individual do animal. Também se foi construido uma pistola com arduino e sensor RFID e uma balança com sensores de pesos para a pesagem mais automatizada.

Aplicação Full Stack com:
- Flask no back-end
- HTML,CSS e JavaScript no front-end
- Mysql como banco

🔗 [Ver Repositório](https://github.com/arthiviz/projeto_suino)
